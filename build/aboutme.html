<This portfolio was created using HTML, CSS, and basic JavaScript. Additionally, I utilized various JavaScript libraries, such as Three.js and GSAP. Three.js is a library that provides a simple and efficient way to create 3D scenes in a web browser. With Three.js, developers can create complex 3D models, animations, and visual effects, and easily integrate them into web applications./>

<On the other hand, GSAP (GreenSock Animation Platform) is a powerful animation library that enables developers to create smooth and engaging animations in a web browser. With its intuitive API, GSAP allows developers to create advanced animations, including timeline-based animations, morphing, and even physics-based animations./>

<By leveraging these technologies, I was able to create a visually engaging and dynamic portfolio. The combination of HTML, CSS, and JavaScript provided the foundation for the structure and design of the portfolio, while Three.js and GSAP added the sophisticated 3D and animation effects. These technologies allowed me to create a unique and immersive experience for the user./>
